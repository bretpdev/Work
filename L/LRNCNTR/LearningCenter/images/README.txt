The images directory is a container for the theme-named directories that will be
available to the application. The actual image files are placed in the theme-named
directories under this directory, because they are expected to be theme-specific.

The theme name is the only dynamic element of the image file path, so the image file
names themselves must be consistent across theme directories. The application uses
a well-defined set of images, so all images must be present, even if they're blank.
